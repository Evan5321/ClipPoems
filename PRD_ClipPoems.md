# 剪贴诗应用 — 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
剪贴诗是一款面向诗歌爱好者、文字创作者和艺术爱好者的创意写作工具。用户通过从语料库中选取文字碎片，在画布上自由拖拽拼贴，创作出具有视觉美感的拼贴诗歌作品。

### 1.2 核心价值
- 降低诗歌创作门槛：通过提供丰富语料激发灵感
- 融合视觉艺术：文字拼贴的过程本身就是视觉创作
- 跨平台创作体验：桌面端精细操作 + 移动端碎片时间创作

### 1.3 目标用户
- 诗歌爱好者（核心用户）
- 创意写作/文字游戏玩家
- 艺术设计学生
- 文学教育场景（课堂创作活动）

## 2. 功能需求

### 2.1 语料库系统

#### 2.1.1 内置语料
| 模块 | 说明 | 优先级 |
|------|------|--------|
| 经典诗歌库 | 收录中外经典诗歌 500+ 首，按作者/流派/时期分类 | P0 |
| 现代文学库 | 现代散文、小说片段 1000+ 条 | P1 |
| 成语/熟语库 | 成语、惯用语、俗语 3000+ 条 | P0 |
| 日常语料库 | 日常对话、描述性语句 2000+ 条 | P1 |
| 主题词包 | 按主题（爱情、自然、城市、时间等）预组词包 | P0 |

#### 2.1.2 语料选取方式
- **推荐分词模式**：从当前文章/诗作中自动分词（基于 jieba），按词性（名词/动词/形容词）分类展示，用户点击选取
- **自主选词模式**：全文浏览语料库，搜索关键词，按分类筛选后选取
- **文章划词模式**：展示完整文章，用户像荧光笔一样划选任意连续文字作为语料碎片

#### 2.1.3 语料碎片属性
每个被选中的语料碎片包含：
- 文本内容
- 来源（文章/诗集名称、作者）
- 词性标签（名词/动词/形容词/短语/句子）
- 字数
- 用户自定义标签

#### 2.1.4 语料扩展（未来）
- **网络爬取**：用户输入 URL，爬取文章内容并清洗为可用语料
- **用户导入**：支持 .txt / .md / .csv 格式导入
- **语料市场**：用户之间分享自建语料包
- **AI 辅助生成**：调用 LLM 根据主题生成候选词句

#### 2.2.1 核心交互
| 功能 | 详细描述 | 优先级 |
|------|---------|--------|
| 拖拽放置 | 从语料面板拖拽碎片到画布，自由位置放置 | P0 |
| 自由拖动 | 画布上已有碎片可任意拖动改变位置 | P0 |
| 缩放旋转 | 双指/快捷键缩放碎片大小，旋转角度 | P0 |
| 层级调整 | 碎片置前/置后/置于顶层/底层 | P0 |
| 删除碎片 | 拖入废纸篓区域 / Delete 键移除 | P0 |
| 多选操作 | Shift+点选 / 框选多碎片，批量移动/删除/调整 | P1 |
| 对齐辅助 | 拖动时显示对齐参考线（与其他碎片对齐） | P1 |
| 吸附网格 | 可选开启网格吸附模式 | P1 |

#### 2.2.2 视觉效果
| 效果 | 实现方式 | 优先级 |
|------|---------|--------|
| 纸片撕裂边缘 | 每个碎片用不规则多边形 clip-path，边缘轻微阴影 | P0 |
| 碎片阴影 | 碎片投射柔和阴影，随层级变化阴影大小 | P0 |
| 拖拽抬起动画 | 拖拽时碎片放大 + 阴影加深 + 略微旋转 | P0 |
| 放置吸附动画 | 放置时弹簧弹性动画 (spring) | P0 |
| 纸片质感 | 碎片背景微米黄/纹理，可选纸张类型 | P1 |
| 飘落效果 | 删除碎片时做飘落消失动画 | P1 |
| 抖动效果 | 长按碎片触发轻微抖动（提示可操作状态） | P2 |

## 3. 非功能需求

### 3.1 性能要求
- 画布支持 50+ 碎片同时存在不卡顿
- 语料库搜索响应 < 200ms
- 应用冷启动 < 3s
- 拖拽交互帧率 > 50fps

### 3.2 兼容性
- 浏览器：Chrome/Firefox/Safari/Edge 最新两个大版本
- 桌面端：Windows 10+ / macOS 11+ / Linux (Tauri)
- 移动端：iOS 15+ / Android 10+ (PWA)

### 3.3 离线能力
- PWA Service Worker 缓存应用资源
- 语料库支持离线使用
- 本地编辑无需网络连接

### 3.4 数据安全
- 所有用户数据默认存储在本地
- 云端同步数据加密传输 (TLS 1.3)
- 用户可随时导出/删除所有数据

### 4.2 交互状态
| 状态 | 视觉反馈 |
|------|---------|
| 拖拽中 | 碎片放大 1.1x，阴影加深，鼠标跟随 |
| 悬停目标 | 若支持吸附，显示高亮吸附位 |
| 选中 | 碎片边框高亮（颜色自定义） |
| 锁定 | 碎片半透明锁图标覆盖，不可拖动 |
| 删除区域 | 底部出现废纸篓区域，拖入高亮变红 |

### 4.3 响应式布局
- ≥1200px：三栏布局（左面板-画布-右面板）
- 768-1200px：两栏布局（画布+浮动面板）
- <768px：全屏画布，面板从侧边滑出/底部弹出

## 5. 数据模型

```typescript
// 语料碎片
interface CorpusFragment {
  id: string;
  text: string;
  source: string;        // 来源文章/诗集
  author?: string;
  posTag?: string;       // 词性
  wordCount: number;
  tags: string[];
  corpusId: string;      // 所属语料库
}

// 画布碎片实例
interface CanvasFragment {
  id: string;
  corpusFragmentId: string;
  text: string;
  position: { x: number; y: number };
  rotation: number;
  scale: number;
  zIndex: number;
  style: {
    fontFamily: string;
    fontSize: number;
    color: string;
    letterSpacing: number;
    lineHeight: number;
    opacity: number;
    direction: 'horizontal' | 'vertical';
  };
  locked: boolean;
}

// 画布
interface Canvas {
  id: string;
  name: string;
  width: number;
  height: number;
  background: {
    type: 'solid' | 'gradient' | 'image' | 'texture';
    value: string;
    opacity: number;
  };
  grid: { enabled: boolean; spacing: number };
  fragments: CanvasFragment[];
  createdAt: string;
  updatedAt: string;
}

// 语料库
interface Corpus {
  id: string;
  name: string;
  type: 'builtin' | 'imported' | 'crawled';
  description: string;
  language: 'zh' | 'en' | 'mixed';
  fragmentCount: number;
  tags: string[];
}
```
